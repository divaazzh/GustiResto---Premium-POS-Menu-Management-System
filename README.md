🍽️ GustiResto - Sistem POS & Manajemen Menu Premium
GustiResto adalah aplikasi kasir (Point of Sale) berbasis web yang dirancang untuk kebutuhan operasional restoran modern dengan estetika kelas atas. Aplikasi ini mengintegrasikan proses transaksi penjualan yang cepat dengan sistem manajemen menu (CRUD) yang dinamis dalam satu antarmuka yang responsif dan elegan.

Proyek ini mendemonstrasikan kemahiran dalam pengembangan frontend menggunakan Vanilla JavaScript dan Tailwind CSS, dengan fokus utama pada manipulasi data di sisi klien tanpa memerlukan refresh halaman.

🛠️ Tech Stack
Bahasa Pemrograman: JavaScript (ES6+)

Styling & Layout: Tailwind CSS (Pendekatan Utility-first)

Ikon & Tipografi: Font Awesome 6, Marcellus & Plus Jakarta Sans

Konsep Inti: Manipulasi DOM Lanjutan & CSS Print Engine

🚀 Fitur Utama
Manajemen Menu (Full CRUD): Kelola inventaris kuliner Anda (Tambah, Lihat, Ubah, Hapus) secara instan tanpa perlu memuat ulang halaman.

Antarmuka Kasir Interaktif: Sinkronisasi keranjang belanja secara real-time, kalkulasi pajak otomatis (10%), dan perhitungan subtotal yang akurat.

Generator Struk Profesional: Struk digital yang bersih dan siap cetak, dioptimalkan menggunakan CSS Media Queries untuk hasil yang presisi.

Simulasi Multi-Pembayaran: Alur kerja terintegrasi untuk berbagai metode transaksi mulai dari Tunai, QRIS, hingga Kartu Debit.

Desain Responsif: Pengalaman pengguna (UI/UX) yang adaptif, dioptimalkan untuk perangkat tablet, laptop, maupun tampilan desktop.
