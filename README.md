# GustiResto---Premium-POS-Menu-Management-System
Aplikasi kasir (Point of Sale) berbasis web yang dirancang untuk kebutuhan operasional restoran modern dengan estetika kelas atas. Aplikasi ini mengintegrasikan proses transaksi penjualan yang cepat dengan sistem manajemen menu (CRUD) yang dinamis dalam satu antarmuka yang responsif dan elegan.
